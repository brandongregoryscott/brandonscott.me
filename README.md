# brandonscott.me

![Netlify Status](https://api.netlify.com/api/v1/badges/df1899a1-e298-4136-9807-25883eb4d895/deploy-status)

Repository for my personal site, [brandonscott.me](https://brandonscott.me).

Built using [Jekyll](https://jekyllrb.com) for static site generation, [Netlify CMS](https://www.netlifycms.org/) for content management and hosted on [Netlify](https://www.netlify.com/).

### Running locally

```
gem install jekyll bundler
bundle install
bundle exec jekyll serve --livereload
```
